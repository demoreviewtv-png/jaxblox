JaxBlox Scripting Guide

Objects process custom JavaScript hooks automatically upon player contact. Code blocks must implement an onTouch(player) structure.

Available API Handlers:

  • player.respawn() — Resets targeted character instantly to spawn location or active checkpoint.
  • player.teleport(x, y, z) — Warps player coordinates instantly to specified vectors.
  • player.setCheckpoint(x, y, z) — Dynamically overrides active spawn/respawn point configurations.

Example Checkpoint Script:

function onTouch(player) {
  // Sets checkpoint to this exact block position
  player.setCheckpoint(0, 5, 10);
}
