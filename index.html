Part Scripting Guide

Implement custom actions safely within parts by executing onTouch(player) context loops.

Global API:

  • player.respawn() - Resets the character instantly.
  • player.teleport(x, y, z) - Teleports player coordinates.
  • player.setCheckpoint(x, y, z) - Overrides character active spawn vector.
